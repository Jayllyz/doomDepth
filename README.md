## Run the project

### Clone

```bash
git clone https://github.com/Jayllyz/doomDepth my-project
cd my-project
```

### Build

```bash
make build
```

### Start

```bash
make start
```

> *Note: make start also run make build.*

### Exec

```bash
make exec
```

### Stop

```bash
make stop
```
